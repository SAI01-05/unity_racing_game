Car Driving Game (Unity)

A 3D car driving simulation game built using Unity, focused on smooth controls and realistic vehicle physics. This project demonstrates core game development concepts such as player movement, physics handling, and camera systems.

## Features
- Realistic car physics using Wheel Colliders
- Smooth acceleration, braking, and steering
- Camera follow system
- Collision detection and environment interaction
- Basic road system using spline tools

## Technologies Used
- Unity Engine
- C#
- Physics System (Wheel Colliders)

## Project Structure
- Scripts: Car controller, camera logic, gameplay mechanics
- Assets: 3D models, textures, materials
- Scenes: Main gameplay scene

## How to Run
1. Open the project in Unity Hub
2. Load the main scene
3. Press Play to start the game

## Controls
- W / Up Arrow: Accelerate
- S / Down Arrow: Brake / Reverse
- A / Left Arrow: Turn Left
- D / Right Arrow: Turn Right

## Purpose
This project was developed to strengthen my skills in Unity game development, particularly in physics-based vehicle systems and gameplay mechanics.
